## :tada: Installation

1. Config enviromental variables
2. Build and run docker with

```
docker-compose up -d
```

## :bulb: Run the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
